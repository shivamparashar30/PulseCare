# 🏥 PulseCare

> **Medico** is the mobile application, while **PulseCare** is the
> open-source project showcasing the architecture and implementation
> behind the platform.

<img width="1086" height="1448" alt="poster" src="https://github.com/user-attachments/assets/941b883f-1983-4677-beea-8aa797687adb" />

## 🚀 Overview

**PulseCare** is a modern healthcare super app built with **React Native
Expo**. The app **Medico** provides doctor appointment booking,
e-pharmacy, diagnostic test scheduling, health records, and secure
payments in a scalable and elegant mobile experience.

## ✨ Features

-   👨‍⚕️ Doctor Appointment Booking
-   💊 Online Pharmacy
-   🧪 Lab Test & Diagnostic Booking
-   📋 Electronic Health Records
-   🏥 Medical Stores
-   💳 Razorpay Integration
-   🔔 Notifications
-   ❤️ Prescription History
-   🌙 Dark Mode
-   📱 Responsive UI

## 🛠 Tech Stack

-   React Native (Expo)
-   TypeScript
-   React Navigation
-   React Query
-   Zustand / Context API
-   Axios
-   React Hook Form
-   Razorpay

## ⚙️ Installation

``` bash

cd PulseCare

npm install

npx expo start
```
